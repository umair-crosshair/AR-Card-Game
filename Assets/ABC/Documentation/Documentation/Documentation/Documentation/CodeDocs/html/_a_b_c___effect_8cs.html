enum ActivateOn
strong

Who the effect should apply and activate on

Entity - Will apply the effect on the entity if hit by the ability. Originator - Will apply the effect on the originator when the ability hits an entity. BothSimultaneously - Will apply the effect simultaneously on both the entity and originator when the ability hits an entity.

ABC_Controller.AbilityGroup Class Reference

Groups together ABC abilities. Allows for additional functionality which focuses on all the abilities in the same group. Such functionality as enabling all abilities in a group at once whilst playing graphics or able to easily change settings for a whole group. Also used to group abilities in the inspector to easily see what you need to change. More...

Detailed Description

Groups together ABC abilities. Allows for additional functionality which focuses on all the abilities in the same group. Such functionality as enabling all abilities in a group at once whilst playing graphics or able to easily change settings for a whole group. Also used to group abilities in the inspector to easily see what you need to change.

Functionality could include changing a setting once that will automatically apply to all abilities in a group, enabling a group during play to change 'character modes' etc
